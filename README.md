# CODSOFT Internship Projects

A collection of full-stack internship projects built during the CODSOFT internship.

---

## Project 1: HireHub

### Brief description
HireHub is a job discovery and hiring platform that helps candidates browse opportunities and apply with their resume, while enabling employers to post jobs, manage applications, and update hiring statuses.

### Key features
- Candidate job browsing with advanced filters (location, salary, job type, experience, remote)
- Resume upload during application
- Save jobs for later
- Employer job management (create/update/close/delete)
- Application status workflow with notifications
- JWT authentication (access + refresh tokens) and secure sessions

### Tech stack
- **Frontend:** React, Vite, Tailwind CSS, Framer Motion, React Hook Form
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Auth/Security:** JWT (access + refresh), bcryptjs, helmet, rate limiting
- **File uploads:** Multer
- **Email:** Nodemailer (Gmail SMTP / Ethereal)

## Project 2: Quizify

### Brief description
Quizify is a modern quiz platform where users can create, discover, and take quizzes. It includes authentication, quiz attempts with timers, scoring, and dashboards with attempt history and leaderboards.

### Key features
- User authentication with secure JWT (cookie-based)
- Create quizzes with categories, difficulty levels, and multiple-choice questions
- Quiz discovery with search/filter/pagination
- Timed quiz attempts with progress tracking and instant scoring
- Dashboard with attempt history and stats
- Leaderboards for top scores
- Responsive UI with modern interactions

### Tech stack
- **Frontend:** React (Vite), Tailwind CSS, React Router, Axios, Toast notifications, Icons
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Auth/Security:** JWT, bcryptjs, helmet, rate limiting
